# wx-AYInstalment

### 微信小程序（爱牙分期）
*******
一个简单的金融申请分期微信小程序

### 基础功能
*******
基础页面包含：列表页、详情页、个人中心页面、提交表单页面、多图片上传编辑、编辑、预览等页面

### 自定义组件
*******
自定义组件包含：表单提交验证文件、自定义吐司弹框、自定义列表tabBar标签切换、自定义pickerview
省市区三级联动
